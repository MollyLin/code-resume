2017_3M_省水募集票選活動

七月 18, 2018 本文总阅读量次

內容

網友投稿並通過審核即可參加抽獎，於期間內公開票選，最高票者即被拍成短片並獲獎品。

負責工作

與設計套版及與後端串資料與表單前端驗證

使用工具與套件

  • Visual Studio use C#
  • Vue.js

功能

  • 接 youtube API 進站自動播放影片並埋 cookie 紀錄 (1 日只顯示 1 次) ; 同頁同 popup 切換 youtube 影片。
  • 線上投稿表單前端驗證 ; 上傳圖片即時編輯使用 Guillotine 套件設定使用戶自訂呈現方式。
  • 運用 Vue.js 的 component event 實作出切換頁籤 (tab) 功能。
  • 接 FB API 實作分享與登入投票功能(截取數據資料傳送給後端資料庫供報表統計及 GA 分析)。
  • 接後端傳回投稿列表並排列或加載於頁面上。/GetArticleList
  • 判斷該用戶 FB 帳號是否符合投票資格 (1 日 3 票) /GetJoinStatus。
  • 配合後端撰寫如投稿作品初次審核成功其他用戶已分享，之後審為失敗時從 FB 分享連結連回網站時的情境。
  • 撰寫如用戶開啟許多瀏覽器分頁且已達投票上限並將 FB 登出，而投票頁未重整時自動跳出登入 FB 帳號之情境判斷。
